# 麻將 / 麻将 májiàng

* 🀀 = dealer / prevailing wind
* 🀁🀂🀃 = winds (south west north)
* 🀅🀆🀇 = dragons (red green white)
* 🀫 = terminal (1 or 9)
* 🀡🀁🀂 = sequence (same suit consecutive)
* 🀅🀅🀅 = triplet / kong 🀅🀅🀅🀅
* 🀄 = dora indicator / bonus tile

## yaku (winning compounds)

# 🀀🀀🀀🀁🀁🀁🀂🀂🀂🀃🀃🀃

* 🀀🀁🀂🀃 + any pair = 十三幺 thirteen orphans
* 🀅🀆🀇 all three = 大三元 big three dragons
* 🀀🀀🀀🀁🀁🀁🀂🀂🀂🀃🀃🀃 + pair = 大四喜 four big winds

## basic calls
* 吃 chī = claim for sequence
* 碰 pèng = claim for triplet
* 杠 gàng = claim for kong
* 和 hú = declare win
